# Delta-Robot-Vision-Based-Sorting-System
esign and development of a 3-DOF Delta Robot prototype demonstrating parallel robot kinematics and high-speed pick-and-place capabilities for industrial automation applications.
This project focuses on the design and fabrication of a 3-DOF Delta Robot prototype inspired by industrial high-speed pick-and-place systems used in manufacturing, packaging, and warehouse automation.

The robot was designed to study parallel kinematics, motion control, workspace analysis, and robotic manipulation while developing practical skills in mechanical design and prototyping.

Objectives
Design a functional Delta Robot mechanism
Understand parallel robot kinematics
Analyze robot workspace
Develop a lightweight robotic structure
Explore industrial automation applications
Features
3 Degrees of Freedom (DOF)
Parallel Kinematic Structure
Lightweight Mechanical Design
High-Speed Motion Capability
Modular Construction
Suitable for Pick-and-Place Operations
